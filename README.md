# Vishawakarma Tent and Building Materials Management Tool

This is a comprehensive management tool for Vishawakarma Tent and Building Materials, designed to handle tent rental and building material inventory, customer ledgers, and invoicing.

## Features

- **Dashboard**: Track daily rental items (Tent, Catering, House Building).
- **Inventory Management**: Manage stock levels and items.
- **Customer Ledger**: Track items given to customers, returns, and pending balances.
- **Invoice Generation**: Create detailed invoices and share via WhatsApp/SMS/Email.
- **History**: View past customer records and transactions.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Database**: Firebase (capable) / Local Demo Mode enabled
- **State Management**: React Context / Hooks

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
